Curriculum map

Six modules blended across design, code, and strategy.

CoociDev Learn_Web is built for designers moving into hybrid UI/front-end roles.

  1. Foundations: research, UX flows, accessibility, HTML/CSS baseline.
  2. Systems thinking: tokens, grids, responsive typography, component APIs.
  3. Modern front-end: semantic HTML, CSS architecture, Three.js integration.
  4. Motion & 3D: micro-interactions, reduced motion patterns, performance.
  5. Shipping & SEO: audits, Lighthouse, content ops, multi-lingual support.
  6. Career studio: portfolio review, interview loops, offer negotiation.
